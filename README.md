# 🌲 Tales of the Forest

A text-based RPG built with React. Hunt creatures, collect loot, level up.

## Play

Live: `https://YOUR_USERNAME.github.io/forest-rpg`

## Run locally

```bash
npm install
npm run dev
```

## Deploy to GitHub Pages

```bash
npm run deploy
```

## Creatures

| Enemy | HP | Loot |
|---|---|---|
| Wild Rabbit | 20 | Rabbit Fur, Lucky Foot, Soft Pelt |
| Field Vole | 30 | Vole Pelt, Tiny Bones, Vole Musk |
| Forest Fox | 50 | Fox Tail, Fox Pelt, Fox Fang |
